# CLAUDE.md

## Project Purpose

This repository is the testing ground for building a simple, static website (HTML, CSS, and JavaScript) one issue at a time. It serves as an isolated environment for iterative development, experimentation, and validation before changes land in the main application.

## What This Repo Contains

- Static HTML, CSS, and JavaScript source files
- Test fixtures and related resources
- Issue-driven incremental changes

## Development Approach

- Build incrementally — one issue at a time
- Keep the stack simple: plain HTML, CSS, and JavaScript (no frameworks unless explicitly required)
- Use modern tooling and browser-native features where possible
- Package manager: **pnpm** (if a package manager is ever needed)

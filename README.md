# force-testing

This repository is the testing ground for the [force-field](https://github.com/SEAjamieD/force-field) application.

## Purpose

`force-testing` contains tests, test fixtures, and related resources used to validate the behavior of the force-field application. It serves as an isolated environment for running test cases, reproducing issues, and verifying new functionality before changes are merged into the main application.

## Related Repository

The application under test lives at: https://github.com/SEAjamieD/force-field

## Usage

Clone this repository alongside the force-field application and use it to run test scenarios against the force-field codebase.

```bash
git clone https://github.com/SEAjamieD/force-testing.git
```
